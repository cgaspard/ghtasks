### Fixes

- **The stray Dock icon is (properly) fixed.** v0.5.1 tried to fix this but didn't fully land: the code that reset the app back to menu-bar-only ran on the wrong thread, so macOS ignored it. It now runs on the main thread, plus a second safeguard re-asserts menu-bar-only whenever macOS reactivates the app (which is what clicking a notification does). GitHub Tasks stays in the menu bar and out of the Dock.

### Install

Download `GH.Tasks_universal.dmg` (signed + notarized), open it, and drag GitHub Tasks to Applications, replacing any earlier copy. Your sources, token, and settings carry over.
