## What's new in v0.4.0

A new **Inbox** that mirrors your GitHub notifications, **clickable desktop notifications** with sound, more row-density options, and a crash fix.

### New

- **Inbox tab.** A new tab that mirrors [github.com/notifications](https://github.com/notifications) directly — your actual notification threads (all reasons, read + unread) in one place. Filter chips match GitHub's inbox (All / Unread / Review requested / Mentioned / Participating / Assigned), read items are dimmed, and a per-row **Done** clears an item. The tab badge shows your **unread count**, just like GitHub. Closed/merged threads that linger in the feed are filtered out automatically.
- **Clickable desktop notifications.** Notifications now use the native macOS notification system: **clicking one focuses the app and jumps to the Inbox**, and they **play the standard notification sound** (respecting your macOS per-app sound setting, Focus, and Do Not Disturb). Notifications fire once per new unread item that needs your response — you won't get pinged twice for something already in your inbox.
- **Optional GitHub sync.** A Settings toggle (**Sync Awaiting with GitHub notifications**, off by default) — when on, opening or clearing an item in GH Tasks also marks its GitHub notification read, so it clears from your real inbox too.
- **Spacious row density.** A fourth density preset above Comfortable, with larger type and more spacing. Row density is Compact / Default / Comfortable / Spacious in Settings → General.

### Changed

- **Window sizes simplified** to **Large** and **Wide** (older sizes map to Large automatically).

### Fixes

- **Fixed a crash that emptied the Issues tab.** Repos with milestoned issues could fail to load with "network error: error decoding response body." Milestone data now decodes correctly (this also unblocked the linked-PR/milestone indicators for those repos).

### Under the hood

- The inbox is sourced from the GitHub Notifications API and classified per GitHub's reasons. Desktop notifications are delivered through a native `UserNotifications` integration (replacing the previous plugin, which couldn't react to clicks). Expanded the automated test suite for the inbox, density, and window-size changes.

### Install

This release is delivered automatically to anyone running v0.1.11+ — look for the update banner or the menu-bar icon change. Manual downloads:

- **macOS:** `GH.Tasks_universal.dmg` (signed + notarized)
- **Windows:** `GH.Tasks_x64-setup.exe` or `GH.Tasks_x64_en-US.msi`
- **Linux:** `GH.Tasks_amd64.AppImage`, `GH.Tasks_amd64.deb`, or `GH.Tasks-1.x86_64.rpm`

Upgrade in place — nothing to reconfigure. (Clickable notifications and sound are macOS features and require the signed app bundle.)
