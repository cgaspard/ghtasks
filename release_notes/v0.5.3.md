### Fixes

- **No more stray Dock icon.** Clicking a desktop notification could promote GitHub Tasks from a menu-bar-only app into a Dock-visible one. It now stays in the menu bar where it belongs — the fix runs on the main thread and re-asserts menu-bar-only whenever macOS reactivates the app.
- **Notifications open GitHub.** Clicking a notification opens the underlying issue or pull request on github.com in your browser, instead of popping the app window.
- **Notifications group by repository.** Multiple pings from the same repo collapse into a single expandable stack in Notification Center, with one place to clear them, instead of piling up as separate alerts.

### New

- **Beta update channel.** Turn on **Settings → General → Receive beta updates** to auto-update from the beta channel — early fixes and features land there before they reach stable. Beta builds are promoted to the matching stable release automatically, so you're never stranded on an old beta. Leave it off to stay on stable.

### Note on dismissing notifications

Whether a notification shows as a **Banner** (fades on its own) or an **Alert** (stays until dismissed) is a macOS setting, not something the app controls. If yours stay on screen, switch it under **System Settings → Notifications → GitHub Tasks → Banners**.

### Install

Download `GH.Tasks_universal.dmg` (signed + notarized), open it, and drag GitHub Tasks to Applications, replacing any earlier copy. Your sources, token, and settings carry over.
