## What's new

### Fixes

- **Notification permission check no longer blocked by CSP.** The `connect-src` directive now allows `ipc:` (and `http://ipc.localhost` for Windows) so the notification plugin can query permission state without the WebView rejecting the IPC call.

### New

- **"Add your first source" CTA on Projects and Issues.** When you have no sources configured for a tab, the empty state now shows a prominent button to add one, plus a short explainer on why sources are needed. Clicking it jumps straight into the Sources editor with the right form pre-opened.
- **Automated release notes.** Release body is now pulled from `release_notes.md` at the repo root. Edit that file and the next `git tag` will ship its contents verbatim — no more post-release `gh release edit` dance.

### Install

- **macOS:** `GH.Tasks_universal.dmg` (signed + notarized, universal — Apple Silicon + Intel)
- **Windows:** `GH.Tasks_x64-setup.exe` or `GH.Tasks_x64_en-US.msi`
- **Linux:** `GH.Tasks_amd64.AppImage`, `GH.Tasks_amd64.deb`, or `GH.Tasks-1.x86_64.rpm`

Upgrade in place — your sources, filters, and cached project data carry over.
