> **This is a beta build.** It ships early and may be rougher than a stable release. To receive builds like this automatically, turn on **Settings → General → Receive beta updates**. Turn it off to return to the stable channel on the next release.

### New

- **Beta update channel.** GitHub Tasks can now auto-update from a beta channel, so early fixes and features reach you before they hit stable. Toggle it in **Settings → General → Receive beta updates**. Beta builds are promoted to the matching stable release automatically, so you're never stranded on an old beta.

### Fixes

- **The stray Dock icon is (properly) fixed.** Clicking a notification could promote GitHub Tasks from a menu-bar-only app into a Dock-visible one. The previous attempt reset the app on the wrong thread, so macOS ignored it; it now runs on the main thread, with a second safeguard that re-asserts menu-bar-only whenever macOS reactivates the app. GitHub Tasks stays in the menu bar.
- **Notifications open GitHub and group by repo.** Clicking a notification opens the issue/PR on github.com; multiple pings from the same repo collapse into one stack instead of piling up.

### Install

Download `GH.Tasks_universal.dmg` (signed + notarized), open it, and drag GitHub Tasks to Applications, replacing any earlier copy. Your sources, token, and settings carry over.
