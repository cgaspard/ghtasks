### Changed

- **The app is now called GitHub Tasks** (it was shown as “GH Tasks”). Same app, clearer name — it updates in place, so there's nothing to reinstall. Your sources, sign-in, and settings all carry over automatically.

### Upgrading

- **Existing installs** update themselves — the auto-updater replaces the app in place and keeps everything. On macOS the app's icon in Finder may still read “GH Tasks” until you next download a fresh copy; everything inside (window, menus, About) shows the new name right away.
- **Fresh downloads:** the installer files are now named `GitHub.Tasks_*` (previously `GH.Tasks_*`).

### Install

Download `GitHub.Tasks_universal.dmg` (signed + notarized), open it, and drag GitHub Tasks to Applications. Your sources, token, and settings carry over.
