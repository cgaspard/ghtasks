### New

- **Inbox filters are now multi-select checkboxes.** The old single-choice Category dropdown became a checkbox popover — the same pattern the Projects tab uses for Status. Pick any combination of categories (they OR together) and see a live count next to each. Empty selection means "All."
- **A dedicated "Unread only" toggle.** Read-state is now its own switch beside the Category picker, independent of which categories you've chosen — so you can, say, show *unread* + *Mentioned* + *Assigned* in one view. Both the category selection and the unread toggle persist across relaunches.
- **You can finally filter subscriptions.** A new **Subscribed & other** category surfaces the reasons that make up most of a busy inbox — `subscribed`, CI activity, releases, state changes — which previously had no checkbox and only ever showed under "All." Check it to isolate them, or check the other four categories to hide the subscription noise entirely.

### Notes

- Because filters persist, the Inbox reopens with whatever category/unread selection you last used. Clear it from the popover's **Clear** button (or untick everything) to return to "All." While any filter is active, infinite scroll pauses — filters operate on already-loaded items, so it won't silently page in items you can't see the effect of.

### Install

- **macOS:** download `GH.Tasks_universal.dmg`, open it, drag GH Tasks to Applications. Signed + notarized — no Gatekeeper prompt.
- **Windows:** `GH.Tasks_x64-setup.exe` (or the `.msi`).
- **Linux:** `GH.Tasks_amd64.AppImage`, `.deb`, or `.rpm`.

Existing users: quit and replace the app; your sources, token, and settings carry over.
