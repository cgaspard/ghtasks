## What's new in v0.1.12

**First release with auto-update.** If you're on v0.1.11, your app will download and install this release automatically the next time it checks (within a few hours of launch, or you can hit **Check again** in the About panel to do it now). Subsequent releases from here forward all ship this way.

### Fixes

- **White flash when clicking the tray icon is gone.** Clicking the menu-bar icon used to briefly show a white rectangle before the dark UI painted. The flash was WKWebView's default first-frame background bleeding through the window transparency. Fixed by enabling `macOSPrivateApi` so the `transparent` code path in [wry#1662](https://github.com/tauri-apps/wry/pull/1662) applies — disables WKWebView's internal white clear without affecting the window's own transparency.
- **Black rectangular frame around the window is gone.** On macOS, enabling the OS-level window shadow (`shadow: true`) draws a rectangular shadow that didn't match the CSS-rounded corners, showing a dark rectangle around the visible rounded shape. Disabled the native shadow and let our CSS `box-shadow` handle it — softer, matches the actual corner geometry.
- **Border outline removed.** Dropped the 1px border on the app container; the drop shadow alone does the separation work, and it looks cleaner against varied wallpapers.

### Install

This version ships automatically to anyone running v0.1.11. Manual download links:

- **macOS:** `GH.Tasks_universal.dmg` (signed + notarized)
- **Windows:** `GH.Tasks_x64-setup.exe` or `GH.Tasks_x64_en-US.msi`
- **Linux:** `GH.Tasks_amd64.AppImage`, `GH.Tasks_amd64.deb`, or `GH.Tasks-1.x86_64.rpm`

Upgrade in place — nothing to reconfigure.
